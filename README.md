# ZoomMemeLord

![PWA](https://img.shields.io/badge/PWA-Installable-purple?logo=googlechrome) ![HTML](https://img.shields.io/badge/Frontend-HTML%2FJS-orange?logo=html5) ![Status](https://img.shields.io/badge/Status-Active-brightgreen)

> **ZoomMemeLord** is a Progressive Web App (PWA) — a fun, meme-themed web experience with service worker support for offline use and installability.

---

## Features

- **Meme-Themed UI** — Fun and entertaining web experience
- **Progressive Web App** — Install directly from the browser
- **Service Worker** — Offline support via `service-worker.js`
- **Lightweight** — Pure HTML and JavaScript, no dependencies

---

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | HTML5 / JavaScript |
| PWA | Service Worker API |

## Quick Start

```bash
git clone https://github.com/hakunaTgl/ZoomMemeLord.git
cd ZoomMemeLord

# Open in browser
open index.html
# Or serve locally
npx serve .
```
